# VenueSync: Intelligent Physical Event Experience

**VenueSync** is a real-time coordination and crowd management platform designed for large-scale sporting venues. It solves the primary challenges of physical events—crowd movement, waiting times, and real-time safety—through predictive analytics and dynamic navigation.

## 🚀 Key Features

### 1. Dynamic Crowd Density Heatmap
Real-time visualization of attendee distribution across the arena. 
- **Live Monitoring**: Tracks "Congested", "Busy", and "Fluid" zones.
- **Predictive Flow**: Forecasts surges before they happen using historical and live data.

### 2. Smart Wayfinding & Coordination
- **Automated Redirects**: Dynamically calculates the fastest paths to concessions or exits.
- **AI Coordination Alerts**: Direct communication with staff to open/close gates based on real-time pressure.
- **Wayfinding Visualization**: AR-ready pathfinding for attendees.

### 3. Real-time Resource Hub
- **Queue Tracking**: Live wait times for food, bathrooms, and merchandise.
- **Staff Optimization**: Predictive staffing suggestions for high-congestion zones.
- **Safety Timeline**: Chronological log of all facility health checks and operational shifts.

## 🛠 Tech Stack
- **Frontend**: React + TypeScript + Vite
- **Styling**: Vanilla CSS (Modern Design System with Glassmorphism)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🎨 UI/UX Design System
- **Theme**: Ultra-dark professional aesthetic with vibrant accent neons (Blue, Purple, Pink).
- **Glassmorphism**: High-blur translucent panels for a premium "Control Center" feel.
- **Micro-interactions**: Pulse animations for active nodes and smooth path transitions for navigation.

## 🚦 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run in development mode**:
   ```bash
   npm run dev
   ```
4. **Access the Dashboard**:
   Navigate to `http://localhost:5173`
